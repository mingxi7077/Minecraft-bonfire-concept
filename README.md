## English

Minecraft-bonfire-concept is the umbrella concept repository for Bonfire, a long-term immersive Minecraft social game project built around NPC interaction, role-based professions, player-driven settlements, logistics systems, and emotional storytelling in a persistent world.

### Project Direction

- Build a social-first Minecraft world instead of a conventional progression-only server.
- Combine NPC relationships, player communities, professions, and long-running city development.
- Keep immersion, emotional continuity, and persistent-world storytelling at the center of the design.

### Current Scope

- World and feature concept notes
- NPC interaction direction
- social and settlement design
- role and profession structure
- technical and presentation experiments attached to the project narrative

### Repository Contents

- `README.md`: project overview
- `Custom.xaml`: local UI or presentation asset
- `cypress.jar`: local supporting artifact currently tracked in the repository

### Status

This repository currently serves as a concept hub and long-term direction marker for the broader Bonfire project.

## 简体中文

Minecraft-bonfire-concept 是 Bonfire 的总概念仓库，用来承载这个长期 Minecraft 沉浸式社交项目的世界观、玩法方向与整体设计意图。

### 项目方向

- 做一个以社交体验为核心，而不是只靠数值推进的 Minecraft 世界。
- 将 NPC 关系、玩家社群、职业系统与长期城市发展结合在一起。
- 让沉浸感、情绪连续性与持续世界叙事成为设计中心。

### 当前覆盖内容

- 世界观与功能概念说明
- NPC 互动方向
- 社交与聚落设计
- 职业与身份结构
- 与项目叙事相关的技术或展示实验

### 仓库内容

- `README.md`：项目总览
- `Custom.xaml`：本地 UI 或展示资源
- `cypress.jar`：当前被纳入仓库的辅助制品

### 当前状态

这个仓库目前主要作为 Bonfire 更大项目体系下的概念中枢与长期方向说明入口。
